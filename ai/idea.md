# 项目目标

做一个本地使用的每日记录系统。

用户通过网页写下当天内容，勾选或维护各类 meta，系统将数据保存到 SQLite，并继续导出 Markdown 和 JSON 以兼容现有归档与展示流程。

---

## 核心功能

- 在线写每日记录
- 保存当天正文、meta 状态和 meta notes
- 自动写入 SQLite
- 自动导出为 Markdown
- 自动更新 meta/字符数 JSON
- 支持 meta 标签统计

---

## 数据原则

- SQLite 为主数据源
- Markdown 为可读归档
- JSON 为展示页兼容产物

---

## 设计取向

- 保持简单文本编辑，不引入复杂编辑器
- 保持单人可维护复杂度
- 后端保持轻量，优先复用现有 Flask agent
- 改造过程尽量兼容现有页面与 heatmap

---

## 非目标

- 不做多人系统
- 不做登录注册
- 不做复杂权限
- 不做云端同步系统
- 不做通用笔记产品
